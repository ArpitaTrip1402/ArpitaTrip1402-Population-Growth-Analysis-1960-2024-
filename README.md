### Population-Growth-Analysis(1960-2024)
Technologies used: Technologies: Python, Pandas, NumPy, Matplotlib, Seaborn
-Analysed a global population dataset containing country-level and aggregate entities.
-Cleaned and filtered the data by identifying and removing non-country aggregates  
-Performed exploratory data analysis to study population trends and growth rates across countries.
-Visualized top populated countries and long-term population changes using line and bar charts, pie 
chart.
-Identified countries with the highest population growth in recent years. -Compared historical population trends between developing and developed nations.
